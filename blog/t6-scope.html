Scope

What is Scope?

In programming the scope of a variable refers the section of code in which a variable exists. Within the section of code that the variable has been created it can be accessed and manipulated in whatever way the programmer designed the code to run. However once you step outside the boundary of this section the variable can no longer be referenced.
However there are ways to make variables available throughout the whole program, In Javascript there are two types of variables Global and Local.

Global Variables

Global variables are variables that are declared outside of a function, these have been written inside the main block of code and can be accessed and manipulated from anywhere within the program giving it a "Global Scope".
As you can see from the example below the myGlobalVar variable has been created outside the functions, this is now a Global variable and can be accessed and changed by any function within the program.

Example:

var myGlobalVar = "This is my global variable!"; // As you can see I have declared my global variable
function firstFunction() { // Here is a function accessing and manipulating the myGlobalVar variable
myGlobalVar = "Global variable accessed from inside first function!";
}
function secondFunction() { // Here is another function accessing and manipulating the myGlobalVar variable
myGlobalVar = "Global variable accessed from inside second function!";
}

Local Variables

Local variables are variables that have been created within a function and have become "local" to that specific function. These variables have a "Local scope" as they can only be accessed and manipulated locally by the function that they have been created by.
Using the same structure as the previous example we can see that myLocalVar has been created inside of my first function. This means that the variable can only be accessed within my firstFunction, when we try and access it from secondFunction we will get a reference error as my second function cannot see the myLocalVar variable.

Example:

function firstFunction() { // Here is a function accessing and manipulating the myGlobalVar variable
var myLocalVar = "This is my local variable!";
.
..
...
console.log(myLocalVar); // This will print out "This is my local variable!" string as it is accessing the variable from inside the function
}

function secondFunction() { // Here is another function accessing and manipulating the myGlobalVar variable
console.log(myLocalVar); // When we try to access a variable that is local to a different function we will get an error as a result
}

What happens if I assign a value to a varables that hasn't been declared?

In the case that you assign a value to a variable that has not been declared yet the variable will automatically become a Global variable.
As you can see from the example myVar variable can be used from both inside and outside the function even though it hasn't been declared as a global variable, this is because it has automatically been done for us by the Javascript interpreter.

Example:

function myFunction() {
myVar = "Hello, World!"

//myVar can be used here
}

//myVar can also be used here

Errors Beware!

The scope of a variable can often be the cause of many problems when learning to code. If you run into a "Uncaught ReferenceError" then double check that all your variables that you are trying to access are global or have been initialized inside of the function.
