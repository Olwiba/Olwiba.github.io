Data-Types

Different types of Data-Types

Here are the 5 different data-types of Javascript:

  • Numbers
  • Strings
  • Booleans
  • Objects
  • Arrays

A Number is a data-type that holds well, You guessed it Numbers! When declaring a number you dont use quotes simply assign a variable to a numeric value. Numbers can be used with arithmetic operators to preform various calculations.

  • Example:
  • var num = 10;

A String is a data-type that is primarily used to hold words/sentences. When you declare a string you must put quotes around the data that you are placing inside of the variable. Stings can hold pretty much any character such as letters A - Z, Numbers and even symbols however stings don't work like numbers do. Anything that is placed inside a sting is considered to the interpreter as a word and standard arithmetic operators don't work the same way as numbers do. Stings can be manipulated in many ways to achieve the desired result such as concatenation, slicing, replacing and many more.

  • Example:
  • var string = "This is my string";

Booleans are a special type of data-type. These only ever have one of two values true or false. I like to think of theses like a light switch, it's either on or off and cant be anything else. This data-type is useful when working with javascript as we can create conditional statements to preform code execution.

  • Example:
  • A good example of a boolean is when i created my game.. I had a playerWon boolean that was set to false at the start of the game and enabled things such as controllers and game play functions. After the game had been won this was set to true and the controls and game play functions stopped execution.
  • var playerWon = false;

An Object data-type is basically a variable holding variables. Pretty much all the data-types in javascript are objects but have been designed to serve a purpose. To declare an Object in Javascript you must put the data inside of curly braces, you can declare both the property title for the specific data and also what is inside it. It's easiest to show you with an example.

  • Example:
  • var myObject = {
  • property1: "Data 1",
  • property2: "Data 2",
  • propertyNumber: 32
  • };

An Array data-type holds multiple values in a single variable. These can be strings, numbers even booleans if you wanted. To declare an Array you use brackets. Inside these brackets you then place all the information you need, you can even leave it blank if you wish to simply just cache a empty array. You can manipulate the array by removing, replacing and adding to the array.

  • Here i created a simple array:
  • var myArray = ["data1", "data2", "data3"];

Say you wanted to access the nth data inside the array you can access it by calling the array and using brackets again placing the data location inside of them.

  • I want to find out what the third item is inside of myArray. Remember that the index starts at 0 so it would be 3 (target) - 1 = 2
  • myArray[2];
  • This would spit out "data3"

Using the access technique I just talked about you can the preform manipulation on the data, Other methods such as finding specific data in an array and replacing it or even "popping" and "pushing" can be done but I'll save that for another day.
